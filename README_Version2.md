# 🥗 Calorie Vision AI

Um aplicativo de análise de alimentos baseado em IA que utiliza visão computacional para estimar o valor nutricional de pratos fotografados.

## 🎯 Funcionalidades

- ✅ **Análise por IA**: Utilize GPT-4 Vision para analisar fotos de alimentos
- ✅ **Estimativa Nutricional**: Calorias, carboidratos, proteínas e gorduras
- ✅ **Histórico Persistente**: Armazene suas análises em banco de dados SQLite
- ✅ **Visualizações**: Gráficos de macronutrientes e calorias por dia
- ✅ **Resumos Estatísticos**: Totais diários, semanais e histórico completo
- ✅ **Retry Automático**: 3 tentativas em caso de falha de conexão
- ✅ **Validação Robusta**: Validação de imagens e dados
- ✅ **Logging Completo**: Sistema de logs para debug e monitoramento
- ✅ **Interface Responsiva**: Design moderno com Streamlit

## 📋 Requisitos

- Python 3.8+
- OpenAI API Key
- Conexão com internet
